ISDI Coders Pasapalabra

Reglas del juego

El juego consiste en acertar las 26 palabras, cada una de las cuales se corresponde con una letra del rosco para la que se ofrece una definición.

Los aciertos se reflejarán en las letras mediante el color verde, mientras que los fallos se mostrarán en color rojo. Las cuestiones "pasapalabra" se mostrarán en color azul.

Para completar el rosco se dispone de un tiempo limitado de 150 segundos. Si el usuario introduce end como respuesta antes de que finalice el tiempo, la partida terminará y no entrará en el ranking.

OPCIONES DE TECLADO:

ENTER: enviar respuesta (si pulsas ENTER sin respuesta, se solicita pasapalabra).

ESPACIO: solicitar pasapalabra (también escribiendo pasapalabra)
